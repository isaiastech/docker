## Comandos docker

* docker ps "Ver os containers passa a flag -t para ver todos"
* rodar ubunto "docker run -it ubuntu"
* docker build -t app .  "Criando Imagem"
* docker images  "Ver imagens"
* docker run -it app sh "Abrindo imagen"
